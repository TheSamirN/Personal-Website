# Samir Nazim — Portfolio

A personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool
- **Tailwind CSS v4** — Styling
- **Framer Motion** — Animations
- **Lucide React** — Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build outputs to the `dist/` folder.

## Features

- Dark/light mode toggle
- Responsive design (mobile-first)
- Animated navigation with 3D flip effects
- Typewriter hero animation
- Interactive location badge with live time
- Smooth scroll between sections
